## 🚀 Quiz App Backend

Backend API cho ứng dụng quiz.

### Tech stack
- Node.js
- Express
- MongoDB

## ⚙️ Setup

### 1. Clone repo
git clone ...

### 2. Cài dependencies
npm install

### 3. Tạo file .env

## 🔐 Environment Variables

Tạo file `.env`:

PORT=any

MONGO_URI=your_mongodb_connection

JWT_SECRET=your_secret

## ▶️ Run
npm start

## APIs: 
POST /login

/register

/users

/quizzes

/quizzes/:quizId

/quizzes/:quizId/questions

/questions

/questions/:questionId
